
1. #ifndef  
Reference : https://www.geeksforgeeks.org/cpp-preprocessor-directives-set-2/  
https://www.techonthenet.com/c_language/directives/ifndef.php
If the MACRO is not defined, it will be defined  
Example  

```
    #define TEST1 100
    #ifndef TEST1
    #define TEST1 111
    #endif 
    --- Output - TEST1 100 ---
 ```
 ```
    #ifndef TEST1
    #define TEST1 111
    #endif 
    --- Output - TEST1 111 ---
```
